Detailed Description

Type that contains the data for a grid. Depending on the need of the application code, the grid may contain integer, real, or cell-by-cell data. This 'generic' data type makes many data validation problems simpler, but at the cost of two unpopulated pointer variables per grid. This implements a coding mechanism comparable to templates in C++.!
