Data Types List
Here are the data types with brief descriptions:
[detail level 12]
controlReads program options from an SWB control file and sets program configuration flags
et_blaney_criddleCalculates potential evapotranspiration by means of the Blaney-Criddle (1950) method, as modified by Jensen and others (1990)
et_hargreavesCalculates potential evapotranspiration by means of the Hargreaves-Samani (1985) method
et_jensen_haiseCalculates potential evapotranspiration by means of the Jensen-Haise (1963) method
et_thornthwaite_matherCalculates potential evapotranspiration by means of the Thornthwaite-Mather (1955, 1957) method
et_turcCalculates potential evapotranspiration by means of the Turc (1961) method
graphCreates plots through calls to the DISLIN library
irrigationProvides support for assessing the effect of irrigation on recharge values by estimating the irrigation required to maintain soil moisture levels for specific crop types
meteorological_functionsProvides support functions used by evapotranspiration and snow routines
modelAllocates memory to store intermediate and final calculation results, keeps track of the model date, reads tabular climate data, and calls the necessary process modules in turn
netcdf_supportProvides support for use of NetCDF files as input for time-varying, gridded meteorlogic data, or output for any SWB-generated variable
t_ellipsoid
rle
runoff_curve_numberUpdates runoff curve numbers and calculates runoff based on those curve numbers
sm_thornthwaite_matherPerforms the actual soil-moisture balance once precip, snowmelt, runoff, and ET have been calculated. Soil moisture for a given soil accumulated potential water loss (APWL) is determined by means of Thornthwaite and Mathers' (1957) tables
snowImplements snow-related calculations (i.e. albedo, snow depth), as well as a simple snow energy balance routine
statsCalculates daily, monthly, and annual statistics for an SWB model run
swb_gridProvides support for input and output of gridded ASCII data, as well as for creation and destruction of grid data structures (defined types)
typesContains types definitions and general utility functions used by the SWB code
approx_equal
assert
chomp

t_graph_configurationType that holds parameters used to create graphics with the DISLIN library
t_grid_collection

t_landuse_lookupContainer for land-use related data
t_model_configurationType that contains model configuration data
t_monthContainer for calendar lookup information
t_netcdf_file
t_ssf_filesContainer for SSF file information
