\subsection{Detailed Description}
Calculates daily, monthly, and annual statistics for an SWB model run. Calculates daily, monthly, and annual statistics for an SWB model run. Also contains routines to:
\begin{DoxyItemize}
\item Extract data from a compressed binary file in order to output grids or plots
\item Write to a PEST $\ast$.ssf file
\item Calculate statistics for a subset of the model domain (i.e. catchment, county)
\item Write daily, annual, and mass balance reports 
\end{DoxyItemize}
